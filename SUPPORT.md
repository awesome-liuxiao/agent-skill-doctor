# Support

Choose the channel that matches the request:

- **Questions and setup help:** use [GitHub Discussions](https://github.com/awesome-liuxiao/agent-skill-doctor/discussions).
- **Confirmed bugs:** open a [bug report](https://github.com/awesome-liuxiao/agent-skill-doctor/issues/new/choose)
  with a synthetic reproduction and sanitized output.
- **Feature proposals:** start in Discussions when the scope is still open; use an issue when the
  expected behavior is concrete.
- **Security or privacy defects:** use private vulnerability reporting as described in
  [SECURITY.md](SECURITY.md).
- **Design-partner participation:** follow [docs/DESIGN_PARTNERS.md](docs/DESIGN_PARTNERS.md).

The development preview is best-effort community support. Stable response-time commitments apply
only to security reports as stated in the security policy.
